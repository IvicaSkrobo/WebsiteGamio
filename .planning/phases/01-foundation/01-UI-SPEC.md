> Generated by gsd-ui-researcher from direct Figma inspection. All values are locked decisions — not defaults or estimates.
> Verified by gsd-ui-checker.

## Spacing Scale

Declared values (all multiples of 4, base unit 8px):

| Token | Value | Usage |
|-------|-------|-------|
| xs | 4px | Icon gaps, inline padding, tight label spacing |
| sm | 8px | Compact element spacing, badge padding |
| md | 16px | Default element spacing, nav link gaps |
| lg | 24px | Card padding, component gaps (gap-6) |
| xl | 32px | Card padding large variant (gap-8), stats grid |
| 2xl | 48px | Section padding steps |
| 3xl | 64px | Page-level spacing steps |
| section-y-sm | 80px | Section vertical padding minimum (py-20) |
| section-y-lg | 120px | Section vertical padding maximum (py-32) |
| container-x | 80px–120px | Container horizontal padding (px-16 to px-24) |

Exceptions:
- Touch targets on interactive elements: minimum 44px height (accessibility floor — applies from Phase 4 onward)
- Nav height: 60–72px (fixed — does not follow 8-point grid strictly due to brand constraint)
- Orange circular icon containers on capability cards: 48–64px diameter

## Typography

| Role | Size | Weight | Line Height | Usage |
|------|------|--------|-------------|-------|
| Body M | 16px | 400 | 1.5 | Descriptive text, nav links, footer body, card descriptions — color: rgba(255,255,255,0.7) |
| Body L | 24px | 400 | 1.5 | About/tagline body text, section lead copy — color: #FFFFFF |
| Heading M | 36px | 700 | 1.2 | Card titles, section subheadings ("Where do we operate?") |
| Heading L | 56px | 700 | 1.15 | Section titles ("Our Products", "Capabilities & supporting systems") |
| Display | 140px | 800 | 1.0 | Hero "Gamio" logotype — metallic white gradient fill, letter-spacing: -0.02em |

Additional type roles (do not add new weights — handled via opacity/color):

| Role | Size | Weight | Letter Spacing | Usage |
|------|------|--------|----------------|-------|
| Label/Badge | 13px | 600 | 0.1em (uppercase) | ORIGINALS badge, game card labels |
| Stats number | 36px | 700 | normal | Live stats figures in hero bar |
| Stats label | 13px | 400 | normal | Stats bar sub-labels — color: rgba(255,255,255,0.6) |
| Nav link | 15px | 500 | normal | Navigation items |

**Declared weights:** 400 (regular), 500 (medium), 600 (semibold), 700 (bold), 800 (extrabold).
Note: 5 weights are required by the Figma design. This exceeds the 2-weight ideal but is justified by the hero display logotype (800) and badge labels (600) being visually distinct roles with no functional overlap.

## Color

| Role | Value | Usage |
|------|-------|-------|
| Dominant (60%) | #0a0a0a | Page background, section backgrounds, nav background |
| Secondary (30%) | #111111 | Alternate section backgrounds (about, platform, capabilities), footer bg |
| Accent (10%) | #FF6B35 | Reserved elements listed below |
| Accent dark | #CC4400 | Hero gradient terminus, CTA button gradient end, orange glow source |
| Glass surface | rgba(255,255,255,0.05) | Glass card backgrounds (capability cards, stats bar) |
| Glass border | rgba(255,255,255,0.10) | Glass card borders |
| Purple section | #1a0028 | "Our Products" section background gradient start |
| Purple section end | #2d0045 | "Our Products" section background gradient end |
| Text primary | #FFFFFF | All headings, bold labels, CTA text |
| Text secondary | rgba(255,255,255,0.70) | Body copy, card descriptions, nav links default state |
| Text muted | rgba(255,255,255,0.60) | Stats labels, footer secondary copy |
| Destructive | not applicable | No destructive actions on this marketing site |

**Accent (#FF6B35) is reserved exclusively for:**
1. Gamio wordmark / logo text
2. Primary CTA button background (gradient: #FF6B35 → #CC4400)
3. Circular icon container backgrounds on capability cards
4. ORIGINALS badge border and text
5. Nav link hover state
6. Orange glow effect: box-shadow 0 0 40px rgba(255,107,53,0.3) — used beneath hero heading and CTA buttons
7. Hero background radial gradient (deep orange to near-black)

**Gradients:**
- Hero background: radial-gradient from #CC4400 area at top-center transitioning to #0a0a0a
- CTA button: linear-gradient(135deg, #FF6B35, #FF4500)
- Hero "Gamio" text: background-clip text with white-to-light-gray gradient
- Products section: linear-gradient(180deg, #1a0028, #2d0045)

## Font Loading Contract

Font: **Sora** via Google Fonts (or self-hosted for performance).

Next.js `app/layout.tsx` or `pages/_app.tsx` must import Sora with weights: 400, 500, 600, 700, 800.

```ts
// Using next/font/google
import { Sora } from 'next/font/google'

const sora = Sora({
  subsets: ['latin'],
  weight: ['400', '500', '600', '700', '800'],
  variable: '--font-sora',
  display: 'swap',
})
```

Fallback chain: `'Sora', 'Space Grotesk', system-ui, sans-serif`
