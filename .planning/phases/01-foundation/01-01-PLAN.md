Run the following commands from the project root to scaffold the project:

```bash
npx create-next-app@latest . --typescript --eslint --app --no-tailwind --no-src-dir --import-alias "@/*" --yes
```

If create-next-app refuses to run in a non-empty directory (CLAUDE.md and .planning exist), move them temporarily, scaffold, then move them back:
```bash
mkdir /tmp/gamio-preserve
mv CLAUDE.md .planning /tmp/gamio-preserve/
npx create-next-app@latest . --typescript --eslint --app --no-tailwind --no-src-dir --import-alias "@/*" --yes
mv /tmp/gamio-preserve/CLAUDE.md /tmp/gamio-preserve/.planning .
rmdir /tmp/gamio-preserve
```

After scaffolding, install Tailwind v4 and GSAP:
```bash
npm install tailwindcss@latest @tailwindcss/postcss@latest postcss
npm install gsap
```

Then configure the three critical config files:

**next.config.ts** — Replace the entire contents with:
```typescript
import type { NextConfig } from 'next'

const nextConfig: NextConfig = {
  output: 'export',
  images: {
    unoptimized: true,
  },
}

export default nextConfig
```

**postcss.config.mjs** — Replace (or create) with:
```javascript
export default {
  plugins: {
    "@tailwindcss/postcss": {},
  },
}
```

Do NOT create a `tailwind.config.js` or `tailwind.config.ts` file. Tailwind v4 uses CSS-first configuration via `@theme` in globals.css (wired in Plan 02).

**app/globals.css** — Replace the scaffold's default content with a minimal Tailwind v4 entry point:
```css
@import "tailwindcss";
```
(Full token wiring happens in Plan 02.)

**app/layout.tsx** — Leave the scaffold default for now (Plan 02 replaces it).

**app/page.tsx** — Replace with minimal placeholder:
```typescript
export default function Home() {
  return (
    <main>
      {/* Sections added in Phase 2 */}
    </main>
  )
}
```

Verify the project starts and builds:
```bash
npm run build
```
The build must complete with zero errors and produce an `out/` directory containing `index.html`.
